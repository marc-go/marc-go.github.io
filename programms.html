Programme

Hier findest du Programmme, die ich zum programmieren oder zur Serververwaltung empfele.

Visual Studio Code (VS Code)

VS Code ist ein Programm von Microsoft fürs Programmieren mit vielen verschiedenen Programmiersprachen. Es ist für Windows, MacOS und Linux verfügbar.

Xammp

Xammp ist ein Localer Webserver für deinen Computer. Du hast also Apache2(Webserver), PHP(Programmiersparche) und MySQL(Datenbankprogramm) auf deinem Computer. Verfügbar für Windows und MacOS.

HTML Editor

HTML Editor ist ein gutes Programm fürs iPad, mit dem man HTML schreiben kann.

Termius

Termius ist ein SSH Client für dein Handy, iPad und Computer.

WinSCP

WinSCP ist ein SFTP Client für Windows.

FTP Manager

FTP Manager ist ein SFTP Client für deine IOS Geräte.

Putty GEN

Ist ein Programm für Windows, mit dem man SSH Schlüssel verwalten kann.
